# JavaScript Practice App - Interactive UI Demo

![JavaScript](https://img.shields.io/badge/JavaScript-ES6-F7DF1E?logo=javascript) ![HTML](https://img.shields.io/badge/HTML-5-E34F26?logo=html5) ![CSS](https://img.shields.io/badge/CSS-3-1572B6?logo=css3) ![Docker](https://img.shields.io/badge/Docker-ready-2496ED?logo=docker)

A classic Tetris game built with vanilla JavaScript, HTML5, and CSS3. Practice project demonstrating DOM manipulation, game loop logic, and keyboard event handling.

## What It Does

- Renders a 20×10 Tetris game board
- Handles keyboard input (arrow keys) for piece movement and rotation
- Game loop for continuous piece descent
- Collision detection with walls and other pieces

> **Note:** This is a practice/skeleton app — game logic (tetrimino shapes, scoring, line clearing) is structured but placeholder-ready for implementation.

## Quick Start

### Run Locally

[![Run Locally](https://img.shields.io/badge/Run%20Locally-make%20up-success?logo=gnu-bash&style=for-the-badge)]()

```bash
make up
```

Open [http://localhost:8081](http://localhost:8081).

### Commands

| Command     | Description                       |
|-------------|-----------------------------------|
| `make up`   | Build and start the app in Docker |
| `make down` | Stop the Docker container         |
